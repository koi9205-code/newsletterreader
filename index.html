Google 로그인 설정이 필요해요

이 앱이 Gmail에 접속하려면 Google Cloud에서 발급받은 클라이언트 ID가 필요합니다. 아래 순서대로 한 번만 설정하면 돼요.

  1. console.cloud.google.com에서 새 프로젝트 생성
  2. "API 및 서비스 → 라이브러리"에서 Gmail API 사용 설정
  3. "OAuth 동의 화면"에서 테스트 사용자로 본인 Gmail 주소 추가 (게시 상태는 "테스트" 유지)
  4. "사용자 인증 정보 → OAuth 클라이언트 ID 만들기" → 애플리케이션 유형: 웹 애플리케이션
  5. 승인된 자바스크립트 원본에 이 사이트 주소 추가 (예: )
  6. 발급된 클라이언트 ID를 app.js 파일 맨 위 CLIENT_ID 값에 붙여넣기

비밀 키가 아니라 클라이언트 ID만 필요합니다. 저장 후 이 페이지를 새로고침하세요.
